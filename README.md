# TastyBytes Snowflake Insights – Data Ingestion

This project simulates a real-world ingestion pipeline for a global food truck network using Snowflake. It includes:

-  Role, warehouse, and schema setup
-  Table creation and S3 stage definition
-  Data ingestion via `COPY INTO`
-  Business queries and profit analysis
-  Semi-structured data parsing with `FLATTEN`

## Tools Used
- Snowflake SQL
- S3 External Stage
- VARIANT + FLATTEN
- GitHub for version control
